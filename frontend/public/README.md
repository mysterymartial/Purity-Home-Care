# Public Assets

## Logo

Place your logo file here as `logo.jpeg`.

**Requirements:**
- Format: JPEG, PNG, or SVG
- Size: 200x200px or larger (square format works best)
- The logo will be displayed in the navbar and footer
- Logo is automatically sized: 64px (mobile) to 80px (desktop)

The logo should represent the Purity Home Care brand and be optimized for web use.
